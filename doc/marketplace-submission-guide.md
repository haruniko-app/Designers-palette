## 3. アプリ説明文（英語）

以下をMarketplace SDK の Detailed Description にコピーしてください。

```
Designer's Palette for Google Slides™ is a professional design toolkit created for designers, marketers, and anyone who wants to create polished presentations efficiently. Built with a designer's perspective to fill the gaps in Google Slides™ native features.

KEY FEATURES

- Designer-Focused UI: Intuitive interface designed from a designer's perspective, not just an engineer's. Every tool is placed where you'd expect it.
- AI Image Extension: Expand images beyond their borders using generative AI - a feature not available in Google Slides™.
- Batch Selection by Attributes: Select all objects with the same color, font, or style in one click - impossible with standard Google Slides™ tools.
- Visual Color Picker with RGB Sliders: Professional-grade color controls with spectrum picker, RGB sliders, and recent color history.
- Precise Alignment & Distribution: Pixel-perfect alignment with multiple reference options (first/last selected, largest/smallest, slide boundaries).
- Style Copy History: Save and reuse object styles across your presentation - go beyond simple copy-paste.

---

IMAGE EDITING

- AI Image Extension: Expand images beyond their borders using generative AI technology. Simply specify the expansion percentage and let AI fill in the surrounding areas naturally.
- AI Upscaling: Enhance image resolution up to 2x with AI-powered upscaling. Perfect for improving low-resolution images.
- Crop Confirmation: Permanently apply image crops to remove hidden portions and reduce file size.
- Color Adjustments: Fine-tune brightness, contrast, saturation, and hue. Individual RGB channel controls for precise color correction.
- Visual Effects: Apply blur, sharpen, grayscale, sepia, invert, and vintage effects to your images.

COLOR MANAGEMENT

- Visual Color Picker: Intuitive spectrum selector with hue slider for selecting any color.
- RGB Sliders: Precise control with individual Red, Green, Blue sliders and numeric input.
- Quick Colors: One-click access to common colors including transparent, white, and black.
- Color Swatches: Pre-defined color palette with multiple shades for consistent design.
- Recent Colors: Automatically saves your recently used colors for quick access.
- Multi-target Support: Apply colors to shape fill, stroke, text foreground, and text highlight.
- Stroke Styling: Control stroke weight and dash style (solid, dotted, dashed).

ALIGNMENT AND LAYOUT

- Object Alignment: Align selected objects to left, center, right, top, middle, or bottom.
- Reference Options: Choose alignment reference from first selected, last selected, largest, smallest, or slide boundaries.
- Even Distribution: Distribute multiple objects evenly in horizontal or vertical direction.
- Precise Spacing: Set exact pixel spacing between objects.

SIZE AND TRANSFORM

- Exact Dimensions: Set precise width and height values for selected objects.
- Aspect Ratio Lock: Maintain proportions when resizing.
- Size Matching: Match width, height, or both dimensions across multiple selected objects.
- Preset Ratios: Quick buttons for common aspect ratios (1:1, 4:3, 16:9, 3:2, 2:1).
- Rotation: Rotate objects by 90 degrees clockwise or counterclockwise.
- Flip: Mirror objects horizontally or vertically.
- Duplication: Duplicate objects horizontally, vertically, or in grid patterns with specified spacing.

TEXT FORMATTING

- Font Selection: Choose from available fonts.
- Font Size: Set precise font size with numeric input.
- Text Styles: Apply bold, italic, underline, and strikethrough formatting.
- Line Spacing: Preset options for 1.0, 1.15, 1.5, and 2.0 line height.
- Paragraph Spacing: Control space before and after paragraphs.

OBJECT MANAGEMENT

- Grouping: Group multiple objects together or ungroup existing groups.
- Layer Order: Move objects to front, back, forward, or backward in the layer stack.
- Batch Selection: Select all objects on the current slide that share the same attribute:
  - Same fill color
  - Same stroke color
  - Same text color
  - Same font family
  - Same font size
  - Same stroke weight

PRIVACY AND SECURITY

This add-on operates entirely within Google Slides™. Your presentation data is processed locally in your browser. AI image features use secure cloud processing, and no data is permanently stored on external servers.

```
Designer's Palette for Google Slides™ は、デザイナー、マーケター、そして洗練されたプレゼンテーションを効率よく作りたいすべての方のための、プロフェッショナルなデザインツールキットです。エンジニア視点ではなく、デザイナー視点で設計された UI で、Google Slides™ 標準機能では実現できない操作を可能にします。

主な特長

- デザイナー視点のUI: エンジニアではなくデザイナーの視点で設計された直感的なインターフェース。すべてのツールが期待する場所に配置されています。
- AI画像拡張: 生成AIを使用して画像の境界を超えて拡張 - Google Slides™ 標準にはない機能です。
- 属性による一括選択: 同じ色、フォント、スタイルのオブジェクトをワンクリックで全選択 - 標準の Google Slides™ ツールでは不可能な操作です。
- RGBスライダー付きビジュアルカラーピッカー: スペクトラムピッカー、RGBスライダー、最近使用した色の履歴を備えたプロ仕様のカラーコントロール。
- 精密な整列・配置: 複数の基準オプション（最初/最後に選択、最大/最小、スライド境界）によるピクセル単位の整列。
- スタイルコピー履歴: オブジェクトのスタイルを保存し、プレゼンテーション全体で再利用 - 単純なコピー＆ペーストを超えた機能です。

---

画像編集

- AI画像拡張: 生成AIを使用して画像の境界を超えて拡張します。拡張率を指定するだけで、AIが周囲を自然に埋めます。
- AI高画質化: AIを使用して画像解像度を最大2倍に向上させます。低解像度の画像の改善に最適です。
- トリミング確定: 画像のトリミングを確定し、非表示部分を削除してファイルサイズを軽減します。
- 色調補正: 明るさ、コントラスト、彩度、色相を調整。RGB個別チャンネルで精密な色補正が可能です。
- ビジュアルエフェクト: ぼかし、シャープ、グレースケール、セピア、反転、ビンテージなどのエフェクトを適用できます。

カラー管理

- ビジュアルカラーピッカー: 色相スライダー付きのスペクトラムセレクターで直感的に色を選択。
- RGBスライダー: 赤、緑、青の個別スライダーと数値入力で精密にコントロール。
- クイックカラー: 透明、白、黒などの一般的な色にワンクリックでアクセス。
- カラースウォッチ: 一貫したデザインのための事前定義カラーパレット。
- 最近使用した色: 最近使用した色を自動保存してすばやくアクセス。
- マルチターゲット対応: 図形の塗りつぶし、枠線、テキスト色、テキストハイライトに色を適用。
- 枠線スタイル: 枠線の太さと線種（実線、点線、破線）を制御。

整列とレイアウト

- オブジェクト整列: 選択したオブジェクトを左、中央、右、上、中央、下に整列。
- 基準オプション: 最初に選択、最後に選択、最大、最小、スライド境界から整列基準を選択。
- 均等配置: 複数のオブジェクトを水平または垂直方向に均等に配置。
- 正確な間隔: オブジェクト間の正確なピクセル間隔を設定。

サイズと変形

- 正確な寸法: 選択したオブジェクトの幅と高さを正確に設定。
- アスペクト比ロック: リサイズ時に比率を維持。
- サイズ揃え: 複数の選択オブジェクト間で幅、高さ、または両方の寸法を揃える。
- プリセット比率: 一般的なアスペクト比（1:1、4:3、16:9、3:2、2:1）のクイックボタン。
- 回転: オブジェクトを時計回りまたは反時計回りに90度回転。
- 反転: オブジェクトを水平または垂直に反転。
- 複製: 指定した間隔でオブジェクトを水平、垂直、またはグリッド状に複製。

テキスト書式

- フォント選択: 利用可能なフォントから選択。
- フォントサイズ: 数値入力で正確なフォントサイズを設定。
- テキストスタイル: 太字、斜体、下線、取り消し線の書式を適用。
- 行間: 1.0、1.15、1.5、2.0の行の高さのプリセットオプション。
- 段落間隔: 段落前後のスペースを制御。

オブジェクト管理

- グループ化: 複数のオブジェクトをグループ化、または既存のグループを解除。
- レイヤー順序: オブジェクトを最前面、最背面、1つ前面、1つ背面に移動。
- 共通選択: 現在のスライド上で同じ属性を共有するすべてのオブジェクトを選択:
  - 同じ塗りつぶし色
  - 同じ枠線色
  - 同じ文字色
  - 同じフォント
  - 同じフォントサイズ
  - 同じ枠線の太さ

プライバシーとセキュリティ

このアドオンはGoogle Slides™内で完全に動作します。プレゼンテーションデータはブラウザ内でローカルに処理されます。AI画像機能は安全なクラウド処理を使用し、外部サーバーにデータが永続的に保存されることはありません。

---

Google Slides™ is a trademark of Google LLC.

## 8. スクリーンショット要件

### サイズ要件
- 推奨: 1280 x 800 ピクセル
- 代替: 640 x 400 または 2560 x 1600 ピクセル
- 最低1枚、最大5枚

### 撮影時の注意事項

含めてはいけないもの:
- OSのメニューバー（Mac: 上部メニュー、Windows: タスクバー）
- ブラウザのアドレスバー、タブ、ブックマーク
- ブラウザのウィンドウ枠
- デスクトップや他のアプリケーション

含めるべきもの:
- Google Slides の編集画面（スライド部分）
- アドオンのサイドバー
- 実際の使用例を示すスライドコンテンツ

### 推奨する5枚の構成

1. メイン画面
   - サイドバー全体が見える状態
   - スライドに複数の要素がある状態
   - 「画像を選択してください」のプレースホルダー表示

2. AI画像拡張
   - 画像を選択した状態
   - プレビュー領域に画像が表示されている
   - 拡張設定のスライダーが見える

3. カラー設定
   - カラーパネルが開いた状態
   - スペクトラムピッカーとRGBスライダーが見える
   - 図形が選択されている状態

4. 整列・配置
   - 複数のオブジェクトが選択されている
   - 整列セクションが開いている
   - 整列後の結果が見える状態

5. 共通選択
   - 同じ色のオブジェクトが複数選択されている状態
   - 共通選択セクションが開いている
   - ボタンにプレビュー色が表示されている

### 撮影手順（Mac）

1. Google Slidesでアドオンを開いた状態にする
2. Command + Shift + 4 を押す
3. スペースバーを押してウィンドウ選択モードにする
4. Google Chromeのウィンドウをクリック
5. 撮影された画像をトリミングしてブラウザ枠を除去
6. 1280 x 800 にリサイズ

### 撮影手順（Windows）

1. Google Slidesでアドオンを開いた状態にする
2. Alt + Print Screen でアクティブウィンドウをキャプチャ
3. 画像編集ソフトでトリミングしてブラウザ枠を除去
4. 1280 x 800 にリサイズ
